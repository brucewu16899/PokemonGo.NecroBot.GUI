#  PokemonGO.NecroBot.GUI
GUI for the Pokemon Go - NecroBot.GUI

## Installation / Usage
 - Download the zip in the latest release
  - https://goo.gl/U2WfFS
 - Unzip everything
 - Change the auth.json and
config.json to suit your needs
- Run PokemonGO.NecroBot.GUI.exe
